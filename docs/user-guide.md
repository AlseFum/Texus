## 开始使用

### 打开文件

在浏览器地址栏中输入文件路径即可打开或创建文件：

```
http://localhost:7123/我的笔记
http://localhost:7123/项目计划
http://localhost:7123/日记/2024-01-01
```

**说明**：
- 如果文件存在，将显示文件内容
- 如果文件不存在，将创建新文件
- 支持中文文件名和路径

### 文件命名规则

- ✅ 支持中文、英文、数字
- ✅ 支持下划线 `_` 和连字符 `-`
- ✅ 支持路径分隔符 `/` 创建层级结构
- ⚠️ 避免使用特殊字符如 `?`、`&`、`%` 等

## 文本编辑器界面

编辑器界面分为四个主要区域：

```
┌─────────────────────────────────────┐
│  标题栏：显示当前文件名              │
├─────────────────────────────────────┤
│  通知栏：显示提示信息（可选）         │
├─────────────────────────────────────┤
│                                     │
│  编辑区：输入和编辑文本内容          │
│                                     │
│                                     │
├─────────────────────────────────────┤
│  底部工具栏：字符统计和操作按钮      │
└─────────────────────────────────────┘
```

### 1. 标题栏

- 显示当前文件名
- 文件名会自动进行 URL 解码，确保中文正确显示

### 2. 通知栏（可选显示）

通知栏用于显示重要提示信息，支持以下类型：

| 图标 | 类型 | 颜色 | 说明 |
|------|------|------|------|
| ℹ️ | 信息 | 蓝色 | 一般提示信息 |
| ⚠️ | 警告 | 橙色 | 需要注意的警告 |
| ❌ | 错误 | 红色 | 错误提示 |
| ✓ | 成功 | 绿色 | 操作成功提示 |
| 📝 | 空文档 | 紫色 | 文档为空的提示 |

**关闭通知栏**：点击通知栏右侧的 `×` 按钮即可关闭。部分通知会自动消失。

### 3. 编辑区

- 主要的文本输入和编辑区域
- 支持多行文本编辑
- 自动调整大小以适应浏览器窗口
- 支持缩进和格式化

### 4. 底部工具栏

包含两个部分：

**左侧**：
- **字符统计**：实时显示当前文本的字符数
- **保存状态**：显示保存进度（保存中/保存成功/保存失败）

**右侧**：
- **保存按钮**：保存当前文本（快捷键 `Ctrl+S`）
- **清空按钮**：清空所有文本内容

## 基本操作

### 编辑文本

1. **输入文本**：直接在编辑区输入文字
2. **换行**：按 `Enter` 键
3. **删除**：使用 `Backspace` 或 `Delete` 键
4. **选择**：鼠标拖拽或 `Shift + 方向键`

### 保存文件

有三种方式保存文件：

1. **点击保存按钮**：点击底部的"保存"按钮
2. **使用快捷键**：按 `Ctrl+S`（Windows/Linux）或 `Cmd+S`（Mac）
3. **保存状态提示**：
   - 保存中：显示蓝色"保存中..."
   - 保存成功：显示绿色"保存成功"
   - 保存失败：显示红色"保存失败"

### 清空文本

1. 点击底部的"清空"按钮
2. 确认提示框中点击"确定"
3. 所有文本将被清空（需要保存才能生效）

⚠️ **注意**：清空操作无法撤销，请谨慎使用！

## 快捷键

### 编辑快捷键

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl+S` | 保存 | 保存当前文本到服务器 |
| `Tab` | 增加缩进 | 增加 4 个空格缩进 |
| `Shift+Tab` | 减少缩进 | 减少 4 个空格缩进 |
| `Ctrl+A` | 全选 | 选择所有文本 |
| `Ctrl+C` | 复制 | 复制选中文本 |
| `Ctrl+V` | 粘贴 | 粘贴剪贴板内容 |
| `Ctrl+X` | 剪切 | 剪切选中文本 |
| `Ctrl+Z` | 撤销 | 撤销上一步操作 |
| `Ctrl+Y` | 重做 | 重做已撤销的操作 |

### 缩进功能详解

编辑器支持智能缩进功能，帮助您更好地组织代码和文本：

#### 单行缩进

- **增加缩进**：按 `Tab` 键，光标所在行会增加 4 个空格
- **减少缩进**：按 `Shift+Tab`，光标所在行会减少 4 个空格

#### 多行缩进

1. 选中多行文本
2. 按 `Tab` 键：所有选中行同时增加缩进
3. 按 `Shift+Tab`：所有选中行同时减少缩进

**示例**：
```
原始文本：
def hello():
print("Hello")

选中后按 Tab：
def hello():
    print("Hello")
```

## 通知栏说明

### 常见通知类型

#### 1. 欢迎信息（信息类型）
```
ℹ️ 欢迎使用文本编辑器！
```
- **颜色**：蓝色
- **何时出现**：首次访问或系统配置的欢迎信息
- **操作**：可以点击 × 关闭，通常会在 3-5 秒后自动消失

#### 2. 空文档提示（空文档类型）
```
📝 当前文档为空，开始编辑吧
```
- **颜色**：紫色
- **何时出现**：打开空文件时
- **操作**：开始输入后可以关闭

#### 3. 保存提示（成功/错误类型）
```
✓ 文件已成功保存
❌ 保存失败：网络错误
```
- **颜色**：绿色（成功）/ 红色（错误）
- **何时出现**：执行保存操作后
- **操作**：自动在 3 秒后消失

#### 4. 警告信息（警告类型）
```
⚠️ 此配置文件修改后需要重启服务
```
- **颜色**：橙色
- **何时出现**：编辑重要文件时的提醒
- **操作**：阅读后可关闭

### 通知栏交互

- **自动关闭**：大多数通知会在 3-5 秒后自动关闭
- **手动关闭**：点击通知栏右侧的 `×` 按钮
- **持久通知**：某些重要警告不会自动关闭，需要手动关闭
- **动画效果**：通知栏的出现和消失都有平滑的动画效果

## 常见问题

### Q1: 为什么我的修改没有保存？

**A**: 请确保：
1. 点击了"保存"按钮或按下 `Ctrl+S`
2. 检查保存状态是否显示"保存成功"
3. 网络连接正常
4. 没有看到红色的"保存失败"提示

### Q2: 如何找到之前保存的文件？

**A**: 在浏览器地址栏输入文件路径：
```
http://localhost:7123/您的文件名
```

### Q3: 编辑器支持哪些浏览器？

**A**: 支持所有现代浏览器：
- ✅ Chrome/Edge（推荐）
- ✅ Firefox
- ✅ Safari
- ✅ Opera

### Q4: 可以上传本地文件吗？

**A**: 目前不支持直接上传文件，但您可以：
1. 打开本地文件
2. 复制内容（`Ctrl+C`）
3. 在编辑器中粘贴（`Ctrl+V`）
4. 保存（`Ctrl+S`）

### Q5: 文件大小有限制吗？

**A**: 理论上没有严格限制，但建议：
- 单个文件不超过 10MB
- 过大的文件可能影响编辑器性能
- 对于大文件，建议分割成多个小文件

### Q6: 如何创建多级目录结构？

**A**: 使用 `/` 分隔符：
```
http://localhost:7123/项目/文档/说明
http://localhost:7123/日记/2024/01/日记
```

### Q7: 能否协同编辑？

**A**: 目前不支持实时协同编辑。如果多人同时编辑同一文件：
- 最后保存的版本会覆盖之前的版本
- 建议通过文件命名或路径区分不同用户的编辑

### Q8: 如何恢复误删的内容？

**A**: 编辑器支持浏览器级别的撤销（`Ctrl+Z`），但仅在当前会话有效。建议：
- 定期保存重要内容
- 对重要文件创建备份副本
- 使用不同的文件名保存不同版本

## 技巧与提示

### 💡 技巧 1: 使用有意义的文件名

**好的文件名**：
```
会议记录-2024-01-15
项目计划-移动端开发
客户反馈-张三-2024-01
```

**不好的文件名**：
```
未命名
文档1
aaa
```

### 💡 技巧 2: 组织文件结构

使用路径创建清晰的文件结构：
```
项目A/需求文档
项目A/技术方案
项目A/会议记录/2024-01-15

项目B/需求文档
项目B/技术方案
```

### 💡 技巧 3: 善用快捷键

熟练使用快捷键可以大大提高编辑效率：
- `Ctrl+S`：随时保存，避免丢失修改
- `Tab` / `Shift+Tab`：快速调整代码缩进
- `Ctrl+A`：快速全选内容

### 💡 技巧 4: 定期保存

- 建议每编辑一段时间就保存一次
- 使用 `Ctrl+S` 快捷键可以轻松保存
- 观察底部的保存状态提示

### 💡 技巧 5: 注意文件编码

- 编辑器使用 UTF-8 编码
- 完全支持中文、日文、韩文等多语言
- 支持 Emoji 表情符号 😊 ✨ 🎉

### 💡 技巧 6: 移动端使用

编辑器支持移动设备访问：
- 响应式设计，自动适配屏幕大小
- 触摸屏友好的界面
- 支持虚拟键盘输入

### 💡 技巧 7: 字符统计

底部实时显示字符数，可用于：
- 统计文章字数
- 控制内容长度
- 查看编辑进度
